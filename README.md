## 🎮 게임 소개

- 현재 튜토리얼 단계 뿐이라 보스 몬스터에게 죽은 이후 리스폰 전까지만 개발됨.(밑에는 현재 목표 기획)
- 몬스터들을 잡으면서 성장을 하며 탐험
- 탐험으로 성장된 자신으로 보스 몬스터들을 잡으며 다님
- 보스 몬스터를 잡으며 새로운 스킬들을 해금함

## 🔍아직 해결하지 못 한 버그들

1. 캐릭터가 카운터 맞으면 데미지 두배로 들어옴
2. 보스전 들어가면 캐릭터가 칼을 든 상태인데 다시 우클릭으로 칼을봅아야함(수정중)
3. 성소 가는 연출중 맨처음에 new뭐시기 나옴
4. 성소 가는 연출중 크로스헤어 나옴
5. 보스 몬스터의 이동 로직이 포지션을 강제로 이동시키는 로직이라 지형지물을 뚫고 플레이어를 추적함(이동로직 수정 예정)
